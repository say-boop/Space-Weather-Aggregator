# 🛰️ Space Weather Aggregator

Дашборд космической погоды. Собирает данные о Солнце, магнитном поле Земли, полярных сияниях и положении МКС из открытых API NASA и NOAA.

---

## ✨ Возможности

- ☀️ **Солнечные вспышки** — классы (A, B, C, M, X), время, источник
- 💨 **Выбросы корональной массы (CME)** — скорость, тип, направление
- 🧲 **Kp-индекс** — текущее состояние магнитного поля Земли с графиком
- 📊 **Архив Kp** — график за 30 дней
- 🌌 **Полярные сияния** — зона видимости, интенсивность
- 🗺️ **Зона сияний на карте** — визуализация на Leaflet
- 🛰️ **Положение МКС** — текущие координаты с картой
- 📍 **Пролёты МКС** — расчёт видимости над городом пользователя
- 🔔 **Уведомления о бурях** — всплывающие алерты при Kp > 5
- 🔄 **Фоновый сбор данных** — обновление каждые 5 минут
- 💾 **Кэширование** — данные сохраняются в JSON
- 📡 **WebSocket** — real-time обновление на фронтенде
- 📄 **Экспорт отчёта** — скачивание HTML-файла с текущими данными
- 🎨 **Тёмная космическая тема** — стильный адаптивный дизайн
- 📱 **PWA** — возможность установить как приложение на телефон/ПК
- 🌍 **i18n** — поддержка русского и английского языков
- ⚡ **Redis** — кэширование данных в памяти (быстрее, надёжнее)
- ☄️ **Астероиды** — ближайшие к Земле объекты (NASA NeoWs)
- 🌤️ **Радиационный пояс** — поток электронов с графиком
- 🌞 **Солнечный ветер** — скорость, плотность, температура
- 🧲 **Bz-компонента** — индикатор вероятности полярных сияний
- 🌍 **3D-глобус** — интерактивная 3D-модель Земли с МКС, сияниями, астероидами
- 📸 **Снимки Солнца** — последние фото с SOHO в разных спектрах (304Å, 195Å, 284Å)
- 🛰️ **Траектория МКС** — путь станции на глобусе

## 🚀 Установка и запуск

### 📋 Предварительные требования

- 🐍 Python 3.12+
- 📦 Poetry
- 🐳 Docker (опционально)
- 🔑 API-ключ NASA (бесплатный: https://api.nasa.gov/)

### ⚙️ Установка

```bash
git clone https://github.com/say-boop/Space-Weather-Aggregator.git
cd Space-Weather-Aggregator
poetry install
```

### 🔐 Переменные окружения

Создайте файл .env в корне проекта:

```text
NASA_API_KEY=ваш_ключ
```

### 💻 Запуск (локально)

```bash
poetry run uvicorn app.main:app --host 0.0.0.0 --port 8000 --reload
```

Открыть в браузере: http://localhost:8000

### 🐳 Запуск (Docker)

```bash
docker compose up --build
```

## 📡 API Endpoints

| Метод | Путь | Описание |
|-------|------|----------|
| `GET` | / | Главная страница дашборда |
| `GET` | /api/data | Все данные из кэша (JSON) |
| `POST` | /api/refresh | Принудительное обновление данных |
| `WebSocekt` | /ws/data | Real-time поток данных (каждые 30с) |
| `GET` | /api/iss/passes?lat=...&lon=... | Пролёты МКС над координатоми |
| `GET` | /api/kp/history | Архив Kp-индекса за 30 дней |
| `GET` | /api/export | Экспорт отчёта в HTML |
