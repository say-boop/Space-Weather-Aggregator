# 🛰️ Space Weather Aggregator

Дашборд космической погоды. Собирает данные о Солнце, магнитном поле Земли, полярных сияниях и положении МКС из открытых API NASA и NOAA.

---

### ✅ Готово:
- ☀️ Солнечные вспышки
- 💨 Выбросы корональной массы (CME)
- 🧲 Kp-индекс с графиком
- 🌌 Полярные сияния
- 🛰️ Положение МКС с картой
- 🔄 Фоновый сбор данных (5 минут)
- 💾 Кэширование в JSON
- 📡 WebSocket real-time
- 🎨 Адаптивный дашборд с тёмной темой
- 🗺️ Интерактивная карта Leaflet
- 📊 График Kp через Chart.js

### 🚧 В разработке:
- 🔔 Уведомления о магнитных бурях (Kp > 5)
- 📍 Расчёт видимости МКС над городом
- 📊 Архив Kp-индекса за 30 дней

## 📡 API Endpoints

| Метод | Путь | Описание |
|-------|------|----------|
| `GET` | `/` | Главная страница дашборда |
| `GET` | `/api/data` | Все данные из кэша (JSON) |
| `POST` | `/api/refresh` | Принудительное обновление данных |
| `WebSocket` | `/ws/data` | Real-time поток данных (каждые 30с) |

## 🚀 Установка и запуск

### 📋 Предварительные требования

- 🐍 Python 3.12+
- 📦 Poetry
- 🐳 Docker (опционально)
- 🔑 API-ключ NASA (бесплатный: https://api.nasa.gov/)

### ⚙️ Установка

```bash
git clone https://github.com/your-username/space-weather.git
cd space-weather
poetry install
```

---

### 🔐 Переменные окружения

Создайте файл .env в корне проекта:

```text
NASA_API_KEY=ваш_ключ
```

---

### 💻 Запуск (локально)

```bash
poetry run uvicorn app.main:app --host 0.0.0.0 --port 8000 --reload
```

### 🐳 Запуск (Docker)

```bash
docker compose up --build
```

---

### 🧪 Тестирование

```bash
poetry run pytest tests/ -v
```
